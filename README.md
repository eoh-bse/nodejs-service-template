# NodeJs Service Template (Typescript)

## Description
A template for a `NodeJs` api service in `Typescript`

## Installation

```bash
$ npm ci
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test
```
